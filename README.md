# TicTacToe-C++
Kleines TicTacToe in c++

Wie spielt man:

1. Lad alle Dateien runter.
2. Starte die ".exe" file.
3. Mit Zahlen wählt man ein Feld aus.
4. Enter um zu malen.
5. Neustarten -> Reboot Datei.
